FLAKE-CTL-LIST(8)
=================

NAME
----

**flake-ctl list** - List registered flake applications

SYNOPSIS
--------

.. code:: bash

   USAGE:
       flake-ctl list

   OPTIONS:
       --user
       --help


DESCRIPTION
-----------

List registered flake applications.

OPTIONS
-------

--user

  List registered flake applications for the calling user

FILES
-----

* /usr/share/flakes

EXAMPLE
-------

.. code:: bash

   $ flake-ctl list
